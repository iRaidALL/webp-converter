# 🚀 SleekWebP Converter

A premium, ultra-fast single-page web application designed to convert images into the modern **WebP** format entirely inside the browser. It helps web developers and e-commerce store owners optimize their image sizes, drastically speeding up their websites and saving bandwidth.

---

## ✨ Features

* **Instant Client-Side Conversion:** Powered by JavaScript Canvas API. Your images never leave your machine (100% private and secure).
* **Premium Dark UI:** Sleek, modern design with smooth glassmorphism effects and drag-and-drop support.
* **Bulk Conversion:** Drop multiple PNG/JPG files at once and convert them in parallel.
* **Smart Size Comparison:** Displays original size (MB) vs optimized WebP size (KB) alongside the exact percentage of space saved (e.g., *Saved 85%!*).
* **Instant Downloads:** Download images individually or export everything at once in a clean `.ZIP` file.

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (Custom animations & responsive layout).
* **Logic:** Pure JavaScript (Client-side Canvas rendering).
* **Deployment:** Hosted with ⚡ on Vercel.
